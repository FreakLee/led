# led
一个Flutter实现的LED跑马灯项目，主要分为四个模块：
* [跑马灯](https://github.com/FreakLee/FlutterLearning/blob/main/Resources/Project_LED/led_01.png)
* [手电筒](https://github.com/FreakLee/FlutterLearning/blob/main/Resources/Project_LED/led_02.png)
* [大字幕](https://github.com/FreakLee/FlutterLearning/blob/main/Resources/Project_LED/led_03.png)
* [设置](https://github.com/FreakLee/FlutterLearning/blob/main/Resources/Project_LED/led_04.png)

涉及到的技术点主要有：
* 状态管理：Riverpod
* 渲染引擎：CustomPainter + Ticker
* 数据持久化：shared_preferences
